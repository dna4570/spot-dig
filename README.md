# 世界ミステリー遺跡マップ

## 概要

世界中の謎や未解明とされる遺跡を地図上で可視化し、クリック操作によって詳細情報をカード形式で表示するフロントエンドアプリです。

ユーザーは地図上のポイントやリストから遺跡を選択し、視覚的に情報を確認できます。
ミステリー・古代文明・宇宙人説などのカテゴリでフィルタリングできる構成を意識しています。

---

## 使用技術

- HTML
- CSS
- JavaScript
- Git / GitHub
- Vercel（予定）

---

## 主な機能

- 世界地図上への遺跡ポイント表示
- クリックによる詳細カード表示
- カード内への画像・テキスト情報表示
- カテゴリフィルター（例：宇宙人説 / 古代文明 / エネルギー）
- 遺跡リストからの選択表示
- UIを重ねたモーダル風デザイン
